# À cœur ouvert – Hébergement Render

## Étapes pour héberger gratuitement

1. Crée un compte GitHub et ajoute tous ces fichiers dans un nouveau dépôt
2. Va sur https://render.com et connecte-toi avec ton GitHub
3. Clique sur "New Web Service" > sélectionne ton repo GitHub
4. Choisis l'environnement **PHP**, le plan **Free**, et clique sur "Create Web Service"
5. Le site sera en ligne à une URL comme https://a-coeur-ouvert.onrender.com

Tu peux consulter toutes les confessions dans le fichier `secrets.txt`
